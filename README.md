# Meet Gami Portfolio

A responsive GitHub Pages portfolio for Meet Gami, built with HTML, advanced CSS, and lightweight JavaScript.

## Sections
- Professional Summary
- Technical Skills
- Professional Experience
- Delivery Highlights
- Projects & Achievements
- Certifications & Education
- Contact

## How to host on GitHub Pages
1. Create a new GitHub repository, for example `meetgami510.github.io` or `portfolio`.
2. Upload `index.html`, `styles.css`, `script.js`, and this `README.md` to the repository root.
3. In GitHub, go to **Settings > Pages**.
4. Select the branch and root folder, then save.
5. Open the generated GitHub Pages URL.

## Customize
- Update links in `index.html` if your LinkedIn or project repositories change.
- Replace text in the Professional Experience or Projects sections as your work grows.
